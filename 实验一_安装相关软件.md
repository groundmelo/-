# 实验一：安装相关软件

## 1. Android Studio 安装

### 1.1 下载 Android Studio

访问 Android Studio 官方网站：https://developer.android.com/studio

选择适合您操作系统的版本进行下载（推荐 4.1 以上版本，以更好支持 LiteRT）。

### 1.2 安装步骤

**Windows 系统：**
1. 双击下载的 `.exe` 文件启动安装向导
2. 选择 "Standard" 标准安装选项
3. 选择安装路径（建议使用默认路径）
4. 点击 "Next" 完成安装
5. 安装完成后，启动 Android Studio

**macOS 系统：**
1. 打开下载的 `.dmg` 文件
2. 将 Android Studio 拖拽到 Applications 文件夹
3. 启动 Android Studio

### 1.3 配置 SDK

首次启动时，Android Studio 会自动下载所需的 Android SDK 组件。

## 2. Jupyter Notebook 和 Python 环境安装

### 2.1 安装 Python

Jupyter Notebook 需要 Python 环境，推荐安装 Python 3.7 以上版本。

**Windows 系统：**
1. 访问 Python 官方网站：https://www.python.org/downloads/
2. 下载 Python 3.9 或更高版本
3. 运行安装程序
4. **重要**：勾选 "Add Python to PATH" 选项
5. 点击 "Install Now"

**验证 Python 安装：**
```bash
python --version
# 或
python3 --version
```

### 2.2 安装 Jupyter Notebook

使用 pip 安装 Jupyter Notebook：

```bash
# 升级 pip
python -m pip install --upgrade pip

# 安装 Jupyter Notebook
pip install jupyter notebook

# 安装常用的机器学习库
pip install numpy pandas matplotlib scikit-learn tensorflow
```

### 2.3 启动 Jupyter Notebook

```bash
# 在命令行中输入
jupyter notebook
```

浏览器会自动打开 Jupyter Notebook 界面。

## 3. Visual Studio Code 安装

### 3.1 下载 Visual Studio Code

访问 VS Code 官方网站：https://code.visualstudio.com/

选择适合您操作系统的版本进行下载。

### 3.2 安装步骤

**Windows 系统：**
1. 双击下载的 `.exe` 文件
2. 阅读并接受许可协议
3. 选择安装路径
4. 根据需要勾选以下选项：
   - 创建桌面快捷方式
   - 添加到 PATH
   - 集成终端支持
5. 点击 "安装" 完成

**macOS 系统：**
1. 打开下载的 `.zip` 文件
2. 将 VS Code 拖拽到 Applications 文件夹

### 3.3 常用扩展推荐

安装 VS Code 后，推荐安装以下扩展：

- **Python** - Python 语言支持
- **Jupyter** - Jupyter Notebook 支持
- **GitLens** - Git 增强工具
- **Live Share** - 实时协作编程
- **Prettier** - 代码格式化

安装扩展方法：
1. 按 `Ctrl+Shift+X`（Windows）或 `Cmd+Shift+X`（macOS）打开扩展面板
2. 搜索扩展名称
3. 点击 "Install"

## 4. 软件验证

安装完成后，建议验证各软件是否正常工作：

### 4.1 验证 Android Studio
启动 Android Studio，检查是否能正常创建项目。

### 4.2 验证 Python 和 Jupyter
```bash
python --version
jupyter --version
```

### 4.3 验证 Visual Studio Code
启动 VS Code，检查是否能正常编辑代码文件。

## 5. 注意事项

1. **系统要求**：确保您的电脑满足各软件的最低系统要求
2. **网络连接**：部分软件安装需要网络连接以下载依赖
3. **磁盘空间**：预留足够的磁盘空间（Android Studio 需要较大空间）
4. **管理员权限**：部分安装可能需要管理员权限
